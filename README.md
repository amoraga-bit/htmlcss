# 🤖 Inteligencia Artificial — Página Web Informativa

Proyecto desarrollado como actividad práctica para la materia de Desarrollo Web. Consiste en una página informativa sobre Inteligencia Artificial, construida con HTML5 y CSS3.
---
## 📄 ¿Qué contiene el proyecto?

- Una página principal con información real sobre la IA: qué es, sus aplicaciones y los modelos más conocidos.
- Un formulario de contacto con cinco campos y validación básica.
- Estilos visuales aplicados desde un archivo CSS externo.
---
## 📁 Archivos
```
htmlcss/
├── index.html      → estructura y contenido de la página
└── css/main.css    → estilos visuales
---
## 🛠️ Tecnologías usadas
- HTML5 con etiquetas semánticas (`header`, `nav`, `main`, `section`, `aside`, `footer`)
- CSS3 con estilos externos
- Sin frameworks ni librerías externas
---
## 🚀 ¿Cómo verlo?
1. Descarga o clona el repositorio.
2. Abre el archivo `index.html` directamente en tu navegador.
3. No requiere servidor ni instalación.
